布林通道（Bollinger Bands）入門：上下軌怎麼解讀

1. 布林通道是什麼

布林通道（Bollinger Bands，簡稱 BB 或布林帶）由美國技術分析師 John Bollinger 於 1980 年代發明，用統計學的「標準差」來判斷股價偏離平均的程度。

核心邏輯：

  • 找一條「中軸」代表「合理價格中心」（通常用 SMA20）
  • 上下各畫一條「標準差」線（通常 ±2σ）
  • 理論上 95% 的時間股價應該落在通道內
  • 突出通道 = 股價偏離「統計常態」，可能反轉或趨勢加速

2. 布林通道怎麼計算

台股標準參數 BB(20, 2)：

中軸 ＝ SMA20（20 日簡單移動平均線）
上軌 ＝ SMA20 ＋ 2 倍標準差
下軌 ＝ SMA20 − 2 倍標準差

「標準差」是統計學中衡量「資料分散程度」的指標。股價波動大 = 標準差大 = 通道寬；股價波動小 = 通道窄。

實際你不用手算，看盤軟體跟領富 AI 都會自動畫好。重點是看通道的「形狀」與「位置」。

3. 三條線的解讀

📊 中軸（SMA20）= 趨勢分界

  • 股價在中軸上方 → 偏多趨勢
  • 股價在中軸下方 → 偏空趨勢
  • 中軸向上彎 = 中期多頭；向下彎 = 中期空頭；橫盤 = 盤整

📈 上軌 = 統計天花板

  • 傳統解讀：股價「碰到」上軌 = 短期超買，可能回檔
  • 強勢解讀：股價「沿著」上軌走 = 強勢趨勢延續

📉 下軌 = 統計地板

  • 傳統解讀：股價「碰到」下軌 = 短期超賣，可能反彈
  • 弱勢解讀：股價「沿著」下軌走 = 弱勢趨勢延續

關鍵：上下軌不是「絕對天花板/地板」，而是「統計上的偏離警示」。要結合趨勢方向看。

4. 收口 vs 開口：擠壓蓄勢

🤏 布林帶「收口」（Squeeze）

上下軌距離縮小、通道變窄。意義：

  • 市場波動降低
  • 多空力量趨於平衡
  • 長期收口後容易爆發大行情（蓄勢待發）

實戰：看到布林帶長期收口時，要準備好「行情啟動」的方向判斷。

📢 布林帶「開口」（Expansion）

上下軌距離擴大、通道張開。意義：

  • 市場波動加劇
  • 有明確趨勢正在形成
  • 方向由「股價突破方向」決定（向上開 = 多頭、向下開 = 空頭）

🎯 經典 Squeeze 突破策略

  1. 等布林帶持續收口 ≥ 10 個交易日（蓄勢）
  2. 觀察股價突破上軌 or 跌破下軌
  3. 突破方向就是新趨勢方向
  4. 進場後跟著趨勢走，直到布林帶再次收口

5. 突破上下軌怎麼用

🔴 突破上軌

不能直接當「賣訊」。要判斷：

  • 量能放大且中軸向上 → 強勢突破，可能續漲（沿軌走）
  • 量能背離（價漲量縮） → 衝高無力，可能修正
  • 突破後迅速跌回 → 假突破，是空頭陷阱

🟢 跌破下軌

不能直接當「買訊」。要判斷：

  • 量能放大且中軸向下 → 強勢破底，可能續跌
  • 跌破後快速拉回中軸 → 假跌破，可能反彈
  • 跌破時 MACD 底部背離 → 反彈訊號加強

6. 布林通道與其他指標搭配

布林 + KD

  • 股價碰上軌 + KD > 80 + KD 死叉 = 強烈短線出場訊號
  • 股價碰下軌 + KD < 20 + KD 黃金交叉 = 強烈短線進場訊號

布林 + MACD

  • 布林開口向上 + MACD 零軸上方多頭 = 確認強勢趨勢
  • 布林收口 + MACD 接近零軸 = 趨勢即將啟動

布林 + 成交量

  • 突破上軌「爆量」= 真突破
  • 突破上軌「縮量」= 假突破機率高

8. 常見問題 FAQ

Q1：布林通道參數一定要用 (20, 2) 嗎？

20 日 + 2 倍標準差是 John Bollinger 的原始建議，也是台股最普遍。短線可用 BB(10, 1.5)；長線可用 BB(50, 2.5)。但混用會讓自己訊號跟市場主流不同步。

Q2：股價可能一直在上軌外面嗎？

會。強勢趨勢時股價可能連續多日在上軌外，叫做「布林帶 walk」。這時候布林通道不能用來抓反轉，反而是強勢延續訊號。要等出現背離或破中軸才考慮出場。

Q3：布林通道適合所有股票嗎？

適合「流動性好、波動規律」的中大型股、ETF。不適合「成交量極小、跳動劇烈」的小型股，因為標準差會被異常單日大幅變動扭曲。

Q4：布林通道跟 ATR 哪個好？

用途不同。布林通道用「標準差」（含方向性、會跟著價格走勢）；ATR 是「平均真實區間」，只看波動大小不看方向。布林通道適合判斷區間，ATR 適合設停損點。兩者互補。

Q5：盤整盤怎麼用布林通道？

盤整盤中布林帶會收口、中軸橫盤。這時候可以做「區間操作」：碰下軌進場、碰上軌出場。但要設好停損（若突破上軌且收口開口，要立刻順勢翻多）。
