💡 為什麼領富 AI 選擇純 Web 服務？

🖥 桌機怎麼用最順手？

方法 1：加入瀏覽器書籤（最簡單）

  1. 用 Chrome / Edge / Firefox / Safari 打開 leadfuai.com
  2. 按 Ctrl+D（Mac: Cmd+D）加入書籤
  3. 下次打開瀏覽器直接點書籤就到了

方法 2：安裝為桌面 PWA（像 APP 一樣）

PWA = Progressive Web App，等於把網頁「裝成桌面捷徑」，開起來像獨立軟體，但本質仍是 Web，不會修改您的系統、不會植入任何東西。

  1. 用 Chrome 或 Edge 打開 leadfuai.com
  2. 網址列右側會出現「⊕ 安裝」圖示，點一下
  3. 確認後，桌面會多一個「領富 AI」捷徑
  4. 從捷徑開啟，就是獨立視窗（沒有瀏覽器分頁、網址列）
💡 PWA 跟「下載軟體」差在哪？
✅ PWA：完全在瀏覽器沙箱、自動同步最新版、無需安裝權限
❌ 傳統軟體：需安裝權限、本機檔案讀寫權限、會留下系統痕跡
