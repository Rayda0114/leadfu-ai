📱 用手機體驗，完全免下載

領富 AI 為 PWA（Progressive Web App）服務，不需從 App Store / Google Play 下載。只要把 leadfuai.com 加到手機主畫面，就會像原生 APP 一樣有獨立圖示、開啟全螢幕、離線可瀏覽歷史資料。

✅ PWA 的好處
· 不占手機儲存空間（< 1 MB）
· 每次開都是最新版，不用更新
· 純瀏覽器沙箱，無法存取通訊錄、相簿、位置等個資
· 跨平台一致體驗（iOS / Android / iPad / 桌機）

🍎 iPhone / iPad 安裝步驟

  1. 用 Safari 打開 leadfuai.com
    ⚠ 必須用 Safari，Chrome / FB 內建瀏覽器不行
  2. 點下方工具列中央的 「分享」按鈕（往上箭頭 ⬆）
  3. 下拉找到 「加入主畫面」 並點一下
  4. 右上角點「加入」
  5. 桌面就會多一個「領富 AI」金色圖示

🤖 Android 安裝步驟

  1. 用 Chrome 打開 leadfuai.com
  2. 瀏覽器底部會自動跳出「安裝領富 AI」提示，點「安裝」
  3. 若沒跳出，點右上角「⋮」選單 → 「加到主畫面」 或 「安裝應用程式」
  4. 桌面就會多一個「領富 AI」圖示，跟其他 APP 一樣使用
⚠ 安全提醒：請務必透過 Chrome 從 leadfuai.com 直接安裝。不要從第三方網站、簡訊連結、LINE 群組訊息下載任何「領富 AI APK 安裝包」 — 那些都是詐騙集團冒用本品牌名稱所發布的惡意檔案。
